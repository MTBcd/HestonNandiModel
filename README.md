# Heston–Nandi Model for OTC Derivatives Pricing

This project implements the **Heston–Nandi GARCH option pricing model** for the valuation of OTC derivatives.  
The model is designed to capture important features of financial markets such as volatility clustering, asymmetric volatility responses, and non-constant conditional variance.

## Overview

The Heston–Nandi model is a discrete-time GARCH framework used for option pricing. Unlike the classical Black–Scholes model, which assumes constant volatility, the Heston–Nandi model allows volatility to evolve over time according to a GARCH process.

This makes the model particularly useful for pricing derivatives in markets where volatility is stochastic and depends on past returns.
